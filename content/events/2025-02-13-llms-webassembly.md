**Talk 1: Gen AI and LLMs in Javascript**
Speaker: Panagiota Mitsopoulou, Senior Frontend Engineer @ Elastic (ex-Microsoft)

An introduction to Generative AI and Large Language Models (LLMs) using Javascript. Practical demonstrations on how to setup and use OpenAI API within Jupyter Notebooks. Learn how to integrate AI in your javascript projects, from setup to execution.

**Talk 2: The Power of Web Assembly in Web Performance**
Speaker: Georgios Giatsidis, Frontend Software Engineer @ BetAdvanced

Explore why WebAssembly is a game-changer for JavaScript developers, its practical use cases, and walk through a hands-on example.
